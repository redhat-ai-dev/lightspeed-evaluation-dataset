# lightspeed-evaluation-dataset

[dataset](./dataset) contains raw and processed Q&A pairs, including the evaluation dataset format to run lightspeed-evaluation tool

**Note** The Q&A dataset is generated with [Ragas: Testset Generation for RAG](https://docs.ragas.io/en/stable/concepts/test_data_generation/rag/)

[rhdh-product-docs](./rhdh-product-docs-1.8-Nov19-2025) is the RHDH RAG docs the dataset is generated from

[categories_rhdh.yaml](categories_rhdh.yaml) is the manual defined topic groups we used to classify Q&A pairs
